## 2. I/O 전문 (`.h`)

**Input `W1701A01_I`** — 헤더(7B) + `irec[]` 그리드 (가변, `grid01_cnt` 개)

| 필드 | 크기 | 설명 (실제 의미) |
|---|---|---|
| `mEnuLvlVl` | 1 | 처리 대상 메뉴레벨 (1/2/3) — 전체 요청 공통 |

| `irec[].mEnuId` | 6 | 메뉴ID (신규행이면 화면에서 미리 채번된 값, 삭제/수정행은 기존 ID) |

| `irec[].hGrnMenuId` | 6 | 상위메뉴ID (레벨>1 일 때 필수) |

| `irec[].mEnuLvlVl` | 1 | 메뉴레벨 (행 단위, `atoi==0` 이면 오류 42031) |

| 필드 | 크기 | 설명 |

## 4. 사용 DB 테이블

| 테이블 | 한글명 | 별칭 | 접근 | 용도 |
|---|---|---|---|---|
| `TB_FXB_CMC010M` | 메뉴목록(메뉴분류) | CMC010M / X / NPLUS1 | SELECT / INSERT / UPDATE / DELETE | 메뉴분류 존재확인, 신규 추가(INSERT), 명·순서·사용여부 수정(UPDATE), 재귀 트리 하위 전체 삭제(DELETE) 및 사용여부 일괄변경 |
| `TB_FXB_CMC011M` | 메뉴화면상세(세부메뉴) | CMC011M | UPDATE / DELETE | 삭제 대상 메뉴의 세부 화면행 삭제, 사용여부 변경 시 세부 화면행 USE_YN 일괄 수정 |
| `TB_FXB_CMC012M` | 메뉴카탈로그(권한별 메뉴리스트) | CMC012M1 / CMC012M2 | UPDATE | 삭제된 메뉴ID 를 사용자·권한별 카탈로그(`MENU_ID_CTLG_CON`) 문자열에서 제거 |
| `DUAL` | (오라클 더미) | — | SELECT | `v_mlst1` 문자열 말미 콤마 제거용 `DECODE/SUBSTR` 계산 |

*DML 요약: CMC010M 은 4종 DML 모두, CMC011M 은 DELETE+UPDATE, CMC012M 은 UPDATE, DUAL 은 SELECT. 트랜잭션은 `W1701A01_F1` 전체를 하나로 묶어 진입함수에서 성공 시 `l_dbcommit()`, 실패 시 `l_dbrollback()`.*

## 5. 로직 / 필터 규칙

- **작업구분 분기 (`switch(prGb)`) 없음.** 행별 `nEwYn` + `wktp` + `ex_flg` + `d_flg` 조합으로 추가/수정/삭제를 판별하는 데이터 기반 방식.
  - `nEwYn=='Y'` → `wktp='1'` → INSERT.
  - `nEwYn!='Y'` → `wktp='2'` → 기존행 UPDATE (+ 사용여부 변경 시 하위 트리 USE_YN 일괄).
  - `nEwYn!='Y' && mEnuId!=' '` 인 행은 "삭제 대상 후보" 로 `v_mlst1` 에 수집 → C1 이 실제 남길/지울 메뉴 계산 후 재귀 삭제.
- **레벨 필터**: `atoi(cIMenuLvel) > 1` 이면 `hGrnMenuId` 필수. 레벨 1 이면 `vHGrnMenuId="000000"` 강제.
- **재귀 트리 (C2/C3)**: `WITH ... UNION ALL` 계층 쿼리로 대상 메뉴의 하위 서브트리 전체 MENU_ID 를 뽑아 삭제/사용여부변경 전파. `DECODE(NPLUS1.MENU_ID,NPLUS1.HGRN_MENU_ID,NULL,...)` 로 자기참조 무한루프 방지.
- **동적 SQL (C1)**: `d_flg` 있을 때만 `strStmt` 조립 → `PREPARE`/`DECLARE ... WITH HOLD`/`OPEN`. `ex_flg` 유무로 `NOT EXISTS` 필터(그리드에 남은 메뉴 제외) 유무가 갈림.
- **CMC011M 삭제/수정**: 레벨값(`cIMenuLvel`/`mMenuLvlVl`)에 따라 `LRDV/MDDV/SMDV_MENU_ID` 중 하나를 `DECODE` 로 매칭.
- **트랜잭션**: 커서/DML 모두 명시 COMMIT/ROLLBACK (TOCOMMIT 아님) — 어느 단계든 실패하면 진입함수에서 전체 롤백.
- **에러 코드 (e_msg.code → mesg, comhdr_o->eflg)**:
  - `32023` — 입력 그리드 건수 ≤ 0
  - `42031` — 메뉴레벨(`mMenuLvlVl`) 이 0/비숫자
  - `42032` — 사용여부(`uSeYn`) 가 'Y'/'N' 아님
  - `42033` — 화면표시순서(`mEnuRpsnSqn`) 빈값
  - `95000` — SQL/커서 오류, 상위메뉴ID 누락(레벨>1) 등 일반 실패
  - `10151` — 메뉴명 중복 (현재 `#if 0` 로 비활성)
  - `00000` — 정상 처리, `comhdr_o->eflg="0"`
