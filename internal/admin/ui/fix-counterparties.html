FIX 카운터파티 등록

mci-edge-fix 가 외부 FIX 카운터파티 (기관 / ECN / MM) 의 Logon 검증에 사용. SenderCompID + Password + Profile (Channel/Site/Tier) + Usid 명시. Password 변경은 mci-edge-fix watch 로 즉시 반영. 새 SenderCompID 추가는 mci-edge-fix 재시작 필요 (Phase C).

등록 / 수정

빈값이면 mci-edge-fix 가 FIX_NEW_ORDER default 사용. ECN 별 alias 등록 시 매매 엔진이 다른 transaction 으로 dispatch.
