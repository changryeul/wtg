## 요청 개요

WTG (Winway Trading Gateway) 개발 환경이 아래 EC2 에서 가동 중입니다.
web/클라이언트 개발자가 접속해서 작업할 수 있도록 **보안그룹 인바운드 규칙
5건 추가**를 요청드립니다. 소요 시간 약 5분, 인스턴스 재시작 불필요합니다.

## 콘솔 작업 경로

EC2 → 인스턴스 → `nh-fxallone-dev` 선택 → 하단 **보안** 탭 → 보안 그룹 클릭
→ **인바운드 규칙 편집** → 위 5건 **규칙 추가** → 저장

## 보안 참고사항

- 해당 서비스들은 평문 HTTP/WS 이므로 **`0.0.0.0/0` (전체 오픈) 은 하지 말아
  주세요.** 반드시 요청된 IP/대역으로만 제한 부탁드립니다.
- 기존 규칙 (SSH 22 등) 은 변경 없이 유지해 주세요.
- 아웃바운드는 변경 불필요합니다.
