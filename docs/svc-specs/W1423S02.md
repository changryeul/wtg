## 3. 구현 흐름 (`.pc`)

```
W1423S02()                       ← 진입 (콜백)
  ├─ prthdr / getmid / getmod    COMHDR + input(mid) + output(mod) 준비
  ├─ E_MSG 기본설정               ltyp="KR", mtyp="S"(FXON), rtyp="M"(Message전송)
  ├─ retc = W1423S02_F1(...)      ← 실제 로직 (작업구분 switch 없음, 무조건 호출)
  ├─ retc==RET_SUCCESS ? eflg="0"+l_dbcommit() : eflg="1"+l_dbrollback()
  │                              (명시적 커밋/롤백, AUTOCOMMIT 아님)
  └─ fnSetMsg(&e_msg, comhdr_o)  → return content->sndl

W1423S02_F1()                    ← 조회 본체
  ├─ getmid/getmod, sndl += W1423S02_O_SZ, memset(mod)
  ├─ 입력 바인딩: yprintf + l_rtrim → iWbcsRlnmAltrNo / iTrnCrcd / iTrnAmt
  ├─ [유효성] iWbcsRlnmAltrNo 길이<1 → e_msg 10201(고객번호 입력) + eflg=1 → RET_FAILURE
  ├─ fnCustInfoQry(&custInfo)    ← 고객약정정보 조회(간접). 실패 → e_msg 10177 + eflg=1
  ├─ [뱅킹조회가능여부] SELECT ... TB_FXB_CMC003D (COM_CD_NM='TRN_CNTL_TIM')
  │      SQL 오류 → e_msg 10118 / vSchYn='N'(조회불가) → e_msg 32713(일자전환 제한)
  ├─ [영업일자] SELECT APLL_BSOP_YMD INTO vBsopYmd FROM TB_FXB_CMG012M (FDM_ID='FRX')
  │      오류 → e_msg 10119 + eflg=1
  ├─ dTrnAmt = atof(iTrnAmt)
  ├─ [1일한도 설정액] SELECT DD1_MXMM_TRAB_AMT FROM TB_FXB_TRC001M
  │      (WBCS_RLNM_ALTR_NO, FX_USER_NO='ALL', FX_PDCD='SPT', CRCD='USD')
  │      NO_DATA_FOUND → e_msg 10231(한도 미등록) / 그외 SQL오류 → 10118
  ├─ [금차소진액 USD환산] iTrnCrcd!='USD' 면 TB_FXB_CMG014F 최신 매매기준율로 환산
  │      → dCurDd1MxmmUgAmt ; USD 면 dCurDd1MxmmUgAmt = dTrnAmt (환산 없음)
  ├─ [소진액/잔액] SELECT NVL(SUM(FX_SDCU_AMT),0), 설정액-소진-금차 INTO 소진액,잔액
  │      FROM ( 미체결 TB_FXB_TRG001L  UNION ALL  체결 TB_FXB_TRG003L+TRG001L )
  │           ×TB_FXB_CMG014F (BASE_CRCD별 최신 매매기준율로 USD 환산)
  │      오류 → e_msg 10310
  ├─ [초과여부] dDd1MxmmUgPsbAmt >= 0 → 'Y' else 'N'   (반전 의심 — §6)
  ├─ mod->* 에 sprintf("%.2f") / zsedit 세팅
  └─ eflg="0", e_msg 91003(정상) → RET_SUCCESS
```

## 5. 로직 / 필터 규칙

- **거래통제시간(뱅킹조회 가능여부)**: `TB_FXB_CMC003D` (COM_CD_NM='TRN_CNTL_TIM', CMCD_VL='0') 의 CON1/CON2 와 현재 HH24MISS 비교. `vSchYn='N'` 이면 일자전환 작업 제한으로 e_msg 32713 후 실패 (§6 조건식 주의).
- **금차 거래금액 USD 환산**: 입력 통화 ≠ USD 이면 `TB_FXB_CMG014F` BASE_CRCD별 최신(BLTN_YMD MAX, BLTN_TOB='1') 매매기준율을 곱함. JPY 는 /100 보정. USD 면 환산 없이 원값 사용.
- **소진액 산정**: 서브쿼리 UNION ALL — (1) `TB_FXB_TRG001L` 미체결 잔량(FX_ORDN_BALN_QTY>0, ORDN_VALD_YMD ≥ 영업일, USE_YN='Y') + (2) `TB_FXB_TRG003L`⋈`TB_FXB_TRG001L` 당일 체결(CNTT_YMD=영업일, CNTT_STTS_DCD='6', USE_YN='Y'). 각 금액을 CMG014F 매매기준율로 USD 환산 후 SUM.
- **잔액**: `설정액(USD) − 소진액(USD) − 금차소진액(USD)`. 한 SELECT 에서 소진액·잔액 동시 산출.
- **트랜잭션**: 조회지만 진입부에서 성공 시 commit / 실패 시 rollback 명시 호출.
- **에러 코드 (e_msg.code → mesg, eflg)**:
  - 10201 고객번호 입력 (eflg=1)
  - 10177 고객정보 미존재 (fnCustInfoQry 실패, eflg=1)
  - 10118 조회 가능시간 조회 오류 / 고객약정 한도조회 SQL 오류
  - 32713 은행 일자전환 작업으로 거래 제한
  - 10119 영업일정보 조회 오류 (eflg=1)
  - 10231 현물환 1일 거래한도 미등록 (NO_DATA_FOUND)
  - 10310 고객별 거래한도(소진/환산) 조회 오류
  - 91003 정상 종료 (eflg=0)
