## 5. 로직 / 필터 규칙

- **처리구분 분기 없음**: `W1119S01` 등과 달리 `prGb`/작업구분 switch 가 없다. 진입 시 무조건 `W1422S01_F1` 단일 조회 경로.
- **누적 소진액 산정**: 플랫폼(원계, EAI `crdgrn_am1/am2`) 누적 + WTG DB 누적(미체결 C + 체결 D)을 합산해 `기존소진액`(설정액/보증금/총액) 산정. USD 환산은 CMG014F 고시환율(JPY 는 /100).
- **은행 미전송분 보정**: F 인라인뷰(`CNTT_STTS_DCD='5'`, `FX_PDCD in('FWD','SWP')`)로 아직 원계 전송 안 된 체결분(`BANK_SEL_CUR`/`BANK_BUY_CUR`)을 계산해, 당행사용금액엔 가산·사용가능금액엔 차감.
- **금차소진액(LmtDcd)**: `'1'`(손실한도설정액)은 io_am12 그대로, `'2'`(현금보증금)은 io_am12×1.1(보증금 가산율) 적용.
- **거래가능 판정**: `손실한도총액(crdgrn_lmt_tram) − 기존총소진 − 금차소진 ≥ 0` → "적정", 미만 → "한도초과" 후 매도·매수 `tr_psb_yn` 강제 'N'. 추가로 각 레그 사용가능금액<0 이면 해당 레그 `io_yn`/`tr_psb_yn` 'N'. 공백 `tr_psb_yn2` 는 'N' 보정.
- **트랜잭션**: `RET_SUCCESS→l_dbcommit / 실패→l_dbrollback`. AUTOCOMMIT 아님.
- **에러/결과 코드**:
  - 성공: `91003` (한도계산결과조회 정상), `eflg=0`.
  - 고객번호 미입력: `10201`, `eflg=1`.
  - 고객정보 없음: `10177`, `eflg=1` (fnCustInfoQry 실패).
  - 통제시간 SELECT 오류: `10118`.
  - 조회불가 시간(일자전환): `32713`.
  - 손실한도 누적 SELECT 오류: `32650` ("대행거래(이종) 체결내역이 존재하지 않습니다." — 메시지 부적절, §6).
  - EAI 호출 실패: `fefnpl19i0` 가 set 한 `e_msg.code` 그대로.
- **결과 전달**: 출력 레코드(27필드) + COMHDR(`eflg`/`e_msg.code`→`mesg`). WTG `/v1/tx` 는 `errn`≠0 이어도 전문 있으면 200, COMHDR 본문 동봉(레거시 규약).
