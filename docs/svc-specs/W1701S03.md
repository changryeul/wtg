## 3. 구현 흐름 (`.pc`)

```
W1701S03()                      ← 진입 (콜백)
  ├─ getmid / getmod            COMHDR + input(mid) + output(mod) 준비
  ├─ E_MSG 기본설정              pname=rkey, ltyp=comhdr_i->ltyp, mtyp="S"(FXON), rtyp="M"
  ├─ switch(comhdr_i->ctyp)     HTS/MTS/WTS/EMP/API/SPI → comhdr_i->chnl = ctyp (그 외 로그만)
  ├─ retc = W1701S03_F1(...)    ← 실제 조회 로직
  └─ retc==RET_SUCCESS ? l_dbcommit() : l_dbrollback()   (명시적 커밋, AUTOCOMMIT 아님)
     └─ fnSetMsg(&e_msg, comhdr_o)  결과코드→메시지 세팅 후 comhdr_o 로그

W1701S03_F1()                   ← 조회 (분기)
  ├─ getmid/getmod, sndl += W1701S03_O_SZ, memset(mod)
  ├─ max_row = MAX_RRB / sizeof(orec[0]),  comhdr_o->rtyp = "G"
  ├─ GRID key 처리 (content->keyc)   K_NEXT→content_get_nkey / K_PREV→content_get_pkey / default→"0"
  │      → v_keyv 세팅 후 l_rtrim   (C1 커서의 > :v_keyv 페이징 키로 사용)
  ├─ 입력 바인딩: yprintf + l_rtrim  (isElectTp/ifXUserDcd/isCreAcsAthrDcd/ibscGrpYn/iuseYn/ilRdvMenuId/imDdvMenuId)
  │      (주석은 "입력항목 유효성 check" 이나 실제 NULL/공백 거부 없음 — §6)
  ├─ if (isElectTp[0]=='1')  DECLARE W1701S03_C1  (권한그룹 목록)
  │      SELECT FX_USER_DCD||SCRE_ACS_ATHR_DCD, FX_USER_DCD, SCRE_ACS_ATHR_DCD,
  │             ATHR_GRP_NM, MENU_ID_CTLG_CON, BSC_GRP_YN, USE_YN
  │        FROM TB_FXB_CMC012M
  │       WHERE FX_USER_DCD LIKE '%'||:ifXUserDcd AND SCRE_ACS_ATHR_DCD LIKE '%'||:isCreAcsAthrDcd
  │         AND USE_YN LIKE '%'||:iuseYn AND FX_USER_DCD||SCRE_ACS_ATHR_DCD > :v_keyv
  │       ORDER BY FX_USER_DCD, SCRE_ACS_ATHR_DCD
  │   else  DECLARE W1701S03_C2  (메뉴 분류 건수 3행: 대분류/중분류/소분류 COUNT UNION ALL)
  │      FROM TB_FXB_CMC011M  (:ilRdvMenuId / :imDdvMenuId 필터, v_keyv 미사용 → C2 페이징 없음)
  ├─ [SQLCODE!=0 after DECLARE] e_msg.code="95000" (return 은 주석처리 — 그대로 진행, §6)
  ├─ OPEN C1/C2   실패 시 e_msg.code="95000", RET_FAILURE
  │      (else-분기의 NO_DATA_FOUND 체크는 OPEN 이 안내는 코드 — dead code §6)
  ├─ for(ii=0; ; ii++)
  │     FETCH C1 → 7컬럼  /  FETCH C2 → :ofXUserDcd(TP), :ofXAthrDcd(CNT) 2컬럼
  │     SQLCODE==NO_DATA_FOUND → break
  │     SQLCODE!=0(기타)       → CLOSE + e_msg.code="95000" + RET_FAILURE
  │     if (ii == max_row) { i_isnext++; break; }   ← 저장 전에 break (해당 행 미저장)
  │     zsedit(mod->orec[ii].*)  각 컬럼 세팅
  │     v_keyv = "%.4s"(strcat(ofXUserDcd, osCreAcsAthrDcd))   ← NEXT KEY 갱신
  ├─ CLOSE C1/C2
  ├─ 결과 판정
  │     ii==0        → e_msg "91001"(자료없음), eflg "0"  (return 주석처리 → 계속)
  │     ii>=max_row  → content_set_nkey(v_keyv), e_msg "53102"(내역 남음), eflg "0"
  │     그 외        → e_msg "91003"(조회완료), eflg "0"
  └─ eflg "0", grid01_cnt=ii, sndl += (ii-1)*sizeof(orec[0]),  RET_SUCCESS
```

## 4. 사용 DB 테이블

| 테이블 | 한글명 | 별칭 | 접근 | 용도 |
|---|---|---|---|---|
| `TB_FXB_CMC012M` | 사용자구분별메뉴목록기본 | `CMC012M` | **SELECT** (커서 `_C1`) | 조회구분='1' 일 때 권한그룹코드/명·메뉴목록내용·기본그룹여부·삭제여부 목록 조회 (사용자구분/화면접근권한/삭제 필터, `FX_USER_DCD||SCRE_ACS_ATHR_DCD` 페이징) |

## 5. 로직 / 필터 규칙

- **모드 분기**: `sElectTp=='1'`(GROUP_AM73) → C1 권한그룹 목록. 그 외(값 무관, `else`) → C2 메뉴 분류 건수. IDCNT_AM71('2') 상수는 정의만 되고 실제 비교에 미사용.
- **C1 필터**: `FX_USER_DCD / SCRE_ACS_ATHR_DCD / USE_YN` 각각 `LIKE '%' || :var` (접미 매칭, 빈 입력이면 전건 매칭). `bscGrpYn` 입력은 트림만 하고 WHERE 에 미사용.
- **C1 페이징**: `FX_USER_DCD||SCRE_ACS_ATHR_DCD > :v_keyv` + `ORDER BY`. `v_keyv` 는 keyc(NEXT/PREV/최초)로 세팅, fetch 루프에서 `"%.4s"(FX_USER_DCD+SCRE_ACS_ATHR_DCD)` 로 갱신 → `content_set_nkey`.
- **C2 특성**: 3행(TP 1/2/3) 고정 반환. `imDdvMenuId` 공백이면 소분류(TP='3') CNT='0' 처리. `v_keyv` 미사용이라 페이징 불가(3행뿐이라 무의미). FETCH 결과 매핑이 특이: `ofXUserDcd←TP`, `ofXAthrDcd←CNT` → 출력 `orec.fXUserDcd`=TP, `orec.fXAthrDcd`=건수.
- **페이징 경계**: `if (ii == max_row) break` 는 **행 저장 전에** 발생하므로 max_row 번째 행은 담기지 않고 다음 페이지로 넘어감(정상 페이징 동작). 이후 `ii>=max_row` 이면 53102(내역 남음).
- **트랜잭션**: `RET_SUCCESS→l_dbcommit / 실패→l_dbrollback`. AUTOCOMMIT 아님.
- **에러/결과 코드**:
  - `91003` 조회완료(정상), `eflg=0`.
  - `91001` 해당 자료 없음(ii==0), `eflg=0` (RET_FAILURE 로 안 빠지고 계속 진행 → 최종 RET_SUCCESS).
  - `53102` 조회 내역 남음(다음 페이지 有), `eflg=0`, NEXT KEY 세팅.
  - `95000` 커서 DECLARE/OPEN/FETCH 오류(내부 로그 "정상 조회" 주석은 오기 §6). OPEN/FETCH 실패 시 RET_FAILURE→rollback.
- **결과 전달**: grid(`grid01_cnt` + `orec[]`) + COMHDR(`eflg`/`e_msg.code`→`mesg`). WTG `/v1/tx` 는 errn≠0 이어도 전문 있으면 200, COMHDR 본문 동봉(레거시 규약).
